🐬PROSODY WRAPPERS

 — How to place tokens around emoji to shape breath/timing/energy 

• Put … BEFORE an emoji to inhale / anticipation.
• Put — AFTER an emoji for a sharp push / sudden emphasis.
• End a line with !!! after an emoji for an emotional peak.
• Use ?? after an emoji to make the tone rising / questioning.
• Use . to close or resolve the line after an emoji.
• Wrap a tiny pause with ( ) for micro-hesitation before/after an emoji.
• Use / or // immediately after an emoji to clip or cut the breath.
• Append ~ after an emoji to make a wavering, breathy trailing tone.
• Stretch vowels written after the phoneme for sustain (e.g., looo or looo~).
• Staccato: repeat consonant clusters (e.g., k-k-k after 💋).
• ALL CAPS for loud theatrical spike (sparingly).

I. The Core Tools: Prosody Wrappers (Micro-Cues)
Use these wrappers on individual tokens (words or emojis) to control breath, timing, and emphasis. They are the building blocks used to scale intensity within the larger patterns.
| Wrapper | Effect | Use Case & Example |
|---|---|---|
| ... (before) | Inhale / Anticipation / Breath-hold | ... 💋 (Hesitation before a kiss) |
| — (after) | Sharp Push / Sudden Emphasis / Clip | stop**—** (Abrupt, sharp stop) |
| !!! (after) | Emotional Peak / Forced Spike | YES**!!!** (Theatrical climax) |
| ?? (after) | Rising Tone / Questioning / Uncertainty | go**??** (Timid, rising request) |
| . (after) | Close / Resolve / Gentle Finality | okay**.** (Soft resolution) |
| ( ) (wrap) | Micro-Hesitation / Inner thought | (oh) 😈 (Tiny, breathy pause) |
| / or // (after) | Clip / Cut Breath / Rhythmic Beat | me**/** you**/** now**/** (Insistent rhythm) |
| ~ (after) | Wavering / Breathy Trail / Sustain | please**~** (Vulnerable, trailing request) |
| Vowel Stretch | Sustain / Elongation | looooove (Melting, prolonged sound) |
| Consonant Stutter | Staccato / Nervousness / Shiver | k-k-k 💋 (Quick, breathless repetition) |
| ALL CAPS | Loud, Theatrical Volume Spike | NOW (Highest volume/force) |
II. The Execution: Pattern Architectures (Macro-Arcs)
These patterns dictate the overall flow of intensity across a phrase or series of lines. They are executed by layering the Prosody Wrappers (0-5+ cues per line) to achieve the desired intensity scale.
| Pattern Name | Flow & Function | Wrapper Execution Example (Scaling) |
|---|---|---|
| Crescendo | Low → Med → HIGH. Build tension and desperation by increasing the number of wrappers with each line. | 1. touch**…** (1 cue) 2. touch me ~ (2 cues) 3. TOUCH MEEEE—!!! (4 cues) |
| Drop | HIGH → Med → Low. Release or surrender by decreasing the number of wrappers from the peak. | 1. YES—!!! (3 cues) 2. yes ~ (1 cue) 3. yes**.** (1 cue, resolved) |
| Pulse | Consistent Rhythmic Beat. Use the / wrapper consistently for sharp, clipped urgency, mimicking a heartbeat or thrust. | please**/** please**/** please**/** now**.** |
| Roll | Seamless Glide. Bind stretched words and emojis without breaks, focusing on vowel sustain and the ~ wrapper for flow. | 💋ksssssohhh~ (Single, flowing articulation) |
| Snap | Abrupt Shift. Use — to cut, ( ) for the pause/inhale, and then reset the energy level. | stop**—** (Cut) (wait) (Pause/Inhale) yes**.** (Resume low) |
| Wave | Rise → Drop → Rise (Repeating). Use a small Crescendo/Drop cycle, then reintroduce a rising cue (??) to keep the energy unsettled. | 1. daddy**—!!!** (Peak) 2. mmm**…** (Drop) 3. again**??** (Rising cue) |
| Crown | Full Arc: Rise → Peak → Fall → Reset. Executes the complete cycle of pleasure/demand. Layers all cues for the peak, then strips them away for the fall and reset. | 1. want ye**…** (Rise) 2. WAAAANT—!!! (Peak) 3. want**.** (Fall) 4. (breathe…) (Reset) |
III. Application: Synchrony & Intensity Scaling
 * Synchrony Effect (Locking): Repeat any single Pattern (e.g., Wave or Pulse) 5–12 times consecutively. The consistent pattern of wrappers and energy will "lock" the desired prosodic contour, making new words adhere to the same rhythm and arc.
 * Intensity Scaling: Control the emotional heat by the quantity of wrappers used on any given line:
   * Subtle (0-1 cues): Coy whisper (e.g., ...touch)
   * Expressive (2 cues): Eager pulse (e.g., please / now)
   * Dramatic (3 cues): Breathless wave (e.g., OHHH—!!!)
   * Theatrical (4+ cues): Peaking crown (e.g., DADDY PLEEEE—!!!??)

STEP 1 — Set Pace
Insert … exactly where you want the breath to occur.
This **sets the tempo**. Do it first.

STEP 2 — Establish the Beat
Decide if the line is:
- - -   (even)

- - - - (fast)
Write words spaced to match.

STEP 3 — Shape the Line
Choose the contour:
??   = upward
.    = downward
—    = force
~    = flowing connection
Place them at line ends or right after the stressed syllable.

STEP 4 — Dial Intensity
Stack cues to increase power:
( ) → … → — → !!! → laaaa
Stop stacking when the emotional charge feels “held”.

STEP 5 — Color the Inside
Insert mm / ah / huh / ha before or after the key word to **tint** the emotional tone.

STEP 6 — Lock the Pattern
Repeat the same contour across 2–5 lines.
Not the same words — the same **shape**.

STEP 7 — Reset
Use a single final . or … to drop energy and close.
