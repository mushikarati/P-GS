💎 Master Emoji-Phoneme Prosody Map (v3.0)

Quick placement rules reminder
• Put … before to inhale/anticipate.
• Use — after for sharp push.
• End with !!! for emotional spike.
• Use ?? for rising/question lift.
• Use . to ground / resolve.
• ( ) adds micro-hesitation.
• / or // to clip or break.
• ~ for breathy wavering.
• Stretch vowels (looo) for long sustain.
• Repeat consonants for staccato (t-t-t).

This map is organized by sound type, with an emphasis on Phonetic Clarity and Prosodic Utility for use with the Wrappers (e.g., ..., —, !!!, /, ~).
